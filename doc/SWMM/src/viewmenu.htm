View Menu

    

The View Menu contains commands for viewing the Study Area Map and the program's toolbars.

 

Command

Description

Dimensions

Sets reference coordinates and distance units for the Map

Pan

Pans across the Map

Zoom In

Zooms in on the Map

Zoom Out

Zooms out on the Map

Full Extent

Redraws the Map at full extent

Query

Highlights objects on the Map that meet specific criteria

Overview

Toggles the display of the Overview Map

Legends

Controls display of the Map legends

Toolbars

Toggles display of toolbars
