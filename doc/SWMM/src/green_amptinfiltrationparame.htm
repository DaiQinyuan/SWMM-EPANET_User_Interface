Green-Ampt Infiltration Parameters

    

Parameter

Description

Suction Head

Average value of soil capillary suction along the wetting front (inches or mm)

Conductivity

Soil saturated hydraulic conductivity (in/hr or mm/hr)

Initial Deficit

Fraction of soil volume that is initially dry (i.e., difference between soil porosity and initial moisture content)

The initial deficit for a completely drained soil is the difference between the soil's porosity and its field capacity. Typical values for all of these parameters can be found in the Soil Characteristics Table.
