Standard Toolbar

    

The Standard Toolbar contains speed buttons for the following commonly used commands:

 

Creates a new project

Opens an existing project

Saves the current project

Prints the currently active window

Copies the current selection to the clipboard or to a file

Finds a specific object on the Study Area Map

Makes a visual query of the Study Area Map

Toggles the display of the Overview Map

Runs a simulation

Displays a run's Status and Summary reports

Creates a new profile plot of simulation results

Creates a new time series plot of simulation results

Creates a new time series table of simulation results

Creates a new scatter plot of simulation results

Performs a statistical analysis of simulation results

Modifies display options for the currently active view

Arranges windows in cascaded style, with the Study Area Map filling the entire display area
