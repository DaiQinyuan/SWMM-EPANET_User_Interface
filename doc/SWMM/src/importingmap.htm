Importing a Map

    

To import the coordinates for a network map stored in a text file:

  1. Select File >> Import >> Map.
  2. Select the file containing the map information from the Open File dialog that appears.
  3. Click OK to replace the current network map with the one described in the file.

This open source version of SWMM uses the GeoJSON format for importing a map. An example GeoJSON file is shown below.

 

The GeoJSON file can be created within other GIS systems by exporting GIS layers as GeoJSON. To be recognized and import properly, the features must have an attribute called 'element_type' and 'element_id'. The 'element_type' should indicate the type of SWMM object, such as 'Junction', 'Outfall', 'Conduit', etc. The 'element_id' should correspond to the SWMM object id. Other attributes are optional, however if an attribute name matches the name of a corresponding SWMM object attribute, that attribute value will be imported.
