Scripting Menu

    

The Scripting Menu provides options for creating and running custom Python scripts within SWMM.

 

The 'Script Editor' provides a simple UI for scripting. Toolbar options allow the user to create a new script, open an existing script, run a script, and save the script to a file.

 

The 'Run Script' option allows a user to select an existing Python script using a file diaolog and then run that existing script.
