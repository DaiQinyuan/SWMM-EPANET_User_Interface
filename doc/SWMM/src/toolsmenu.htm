Tools Menu

    

The Tools Menu contains commands used to configure program preferences, Study Area Map display options, and external add-in tools.

 

Command

Description

Program Preferences

Sets program preferences, such as font size, confirm deletions, number of decimal places displayed, etc.

Map Display Options

 

Sets appearance options for the Map, such as object size, object annotation, flow direction arrows, and back-ground color

Configure Tools

Adds, deletes, or modifies external add-in tools
