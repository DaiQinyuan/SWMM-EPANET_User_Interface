To generate compiled help, open a command prompt, cd to this folder, then:

Generate compiled help from a qt help project:  C:\Qt\Qt5.5.1\5.5\msvc2013_64\bin\qhelpgenerator swmm.qhp -o swmm.qch

(Optional: Register compiled help with Qt Assistant:  assistant -register swmm.qch)

Generate qt help collection from a qt help collection project:  C:\Qt\Qt5.5.1\5.5\msvc2013_64\bin\qcollectiongenerator swmm.qhcp -o swmm.qhc


Then to run in Qt Assistant from command line:

assistant -collectionFile swmm.qhc
