Editing Controls

To edit either Simple or Rule-Based Controls:

  1. Select Controls from the dropdown list of the Data Browser.
  2. Select Simple or Rule-Based from the Data Browser.
  3. Click the Browser's Edit button .
  4. Type and edit your control statements in the Controls Editor dialog box that appears.
  5. Click the Editor's OK button to save your edits.
