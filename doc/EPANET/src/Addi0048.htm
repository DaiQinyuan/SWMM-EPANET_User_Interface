Adding a Node

To add a Node using the Map Toolbar:

  1. Click the button for the type of node (junction , reservoir , or tank ) to add from the Map Toolbar if its not already depressed.
  2. Move the mouse to the desired location on the map and click.

    To add a Node using the Browser:

  1. Select the type of node (junction, reservoir, or tank) from the dropdown list of the Data Browser
  2. Click the Browser's Add button .
  3. Enter map coordinates with the Property Editor (optional).
