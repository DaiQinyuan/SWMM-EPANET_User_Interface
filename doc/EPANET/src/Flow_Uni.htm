Flow Units

The units in which nodal demands and link flow rates are expressed determine whether US or metric units are used for all other quantities. Choosing liters or cubic meters causes all other units to be SI metric, otherwise US customary units apply.

Flow units are part of the project's Hydraulic Options. To change them, see Setting Analysis Options or Setting Project Defaults.

Use caution when changing flow units as it might affect all other data supplied to the project.
