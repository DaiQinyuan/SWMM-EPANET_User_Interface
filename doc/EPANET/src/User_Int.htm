User Interface Enhancements

  1. A Visual Editing system has been implemented for using point-and-click methods to add and modify system elements directly on the network map.
  2. Special editors have also been provided for entering data for node/link properties, junction demands, time pattern factors, pump and other curves, control rules, and analysis options.
  3. A Group Editing feature has been added which allows the user to select objects within an irregular shaped portion of the network and apply such editing operations as "For all pipes with Tag equal to CLASS-A replace roughness with 110."
  4. Map labels can be used as "meters" to dynamically display analysis results at selected nodes or links on the map.
  5. A Map Query function can be called on to highlight those nodes or links on the map that meet a specific user-defined criterion (e.g., it will show all nodes where the pressure is below 20 psi and hide the rest).
  6. Links can be represented as polylines with intermediate points that can be moved around to add shape and curvature.
  7. Enhanced map display options are available including the ability to make node or link size be proportional to property value, to hide symbol, flow arrow and notation display until a certain map scale is zoomed in to, and to display a backdrop map image (e.g., a street map) behind the pipe network map.
  8. The map display can be animated to move either forward or backward in time at a user-adjustable speed.
  9. Energy Reports can be generated that list energy consumption and cost for all pumps in the system.
  10. A Reaction Report can be generated that shows what fraction of network-wide water quality reactions occur in the bulk flow, at the pipe wall, and in storage tanks.
  11. Calibration Reports can be produced that statistically evaluate a model's goodness of fit to measured data.
  12. Profile, Frequency and System Flow plots have been added to Time Series and Contour plots for graphical display of simulation results.
  13. Multiple nodes or links can be plotted on the same time series graph.
  14. An enhanced set of formatting options is available to make graphs of publication quality.
  15. The tabular display of results has been enhanced by allowing the user to choose which quantities should appear in a table, which quantity should be sorted on, and how the table entries should be filtered with respect to one or more conditions.
  16. The number of decimal places displayed for each computed parameter can be set by the user.
  17. The current set of reports, graphs, and tables being displayed are updated automatically after a new network analysis is made.
  18. The network map can be saved to disk as map image, GeoJSON, shapefile, or CSV files.
