Standard Toolbar

The Standard Toolbar contains speed buttons for commonly used commands as follows:

 

Opens a new project

 

Opens an existing project

 

 

Saves the current project

 

 

Prints the currently active window

 

 

Copies selection to clipboard or to file

 

 

Deletes currently selected item

 

 

Finds a specific item on the map

 

 

Runs a simulation

 

 

Runs a visual query on the map

 

 

Creates a new graph view of results

 

 

Creates a new table view of results

 

 

Modifies options for the currently active view
