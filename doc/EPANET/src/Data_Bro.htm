Data Browser

The Data Browser is accessed from the Project tab on the Project Explorer window. It gives access to the various objects, by category (Junctions, Pipes, etc.) that are contained in the network being analyzed. The buttons at the top of the object explorer are used to add, delete, and edit these objects.
