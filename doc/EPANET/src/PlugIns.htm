Plugins Menu

The Plugins menu lists user-developed plugins for EPANET.

The plugins folder installed with EPANET contains example/demo user plugins. Plugins are dynamically loaded into the 'Plugins' menu on the main window and they are checkable as toggles to control menu creation.

Three sample plugins are included with EPANET to demonstrate different types of user plugins:

 

     The ImportExportGIS plugin adds a new menu to the UI, with options to import and export the current map to GIS files. This plugin has the same functionality as the 'File:Import' and 'File:Export' menus, but it is included here to demonstrate the plugin capabilities.

 

     The MyCalculations plugin demonstrates how a user could create a plugin that runs without creating its own menu item.

 

     The Summary plugin creates its own menu item with three options to summarize the contents of the existing project.
