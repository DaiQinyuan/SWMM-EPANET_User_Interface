Edit Menu

The Edit Menu contains commands for editing.

Command

Description

Select Object

Allows selection of an object on the map

Select Vertex

Allows selection of link vertices on the map

Select Region

Allows user to outline a region on the map for selecting multiple objects

Select All

Makes the outlined region the entire viewable area of the map

Edit Object

Makes the current selected object available for editing in the property editor

Delete Object

Deletes the current object

Undo

Reverts the most recent change to the project

Redo

Restores the most recent change to the project
