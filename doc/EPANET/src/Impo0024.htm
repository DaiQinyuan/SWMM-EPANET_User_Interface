Importing a Network Map

To import the coordinates for a network map stored in a text file:

  1. Select File >> Import >> Map.
  2. Select the file containing the map information from the Open File dialog that appears.
  3. Click OK to replace the current network map with the one described in the file.

This open source version of EPANET uses the GeoJSON format for importing a map. An example GeoJSON file is shown below.

The GeoJSON file can be created within other GIS systems by exporting a GIS layers as GeoJSON. To be recognized and import properly, the features must have an attribute called 'element_type' and 'element_id'. The 'element_type' should indicate the type of EPANET object, such as 'Junction', 'Pump', 'Tank', 'Pipe', etc. The 'element_id' should correspond to the EPANET object id. Other attributes are optional, however if an attribute name matches the name of a corresponding EPANET object attribute, that attribute value will be imported.
