Creating a Calibration Report

To create a Calibration Report:

  1. First make sure that Calibration Data for the quantity being calibrated has been registered with the project (see Registering Calibration Data).
  2. Select Report >> Calibration.
  3. In the Calibration Report Options form that appears

    4.      select a parameter to calibrate against

         select the measurement locations to use in the report

  4. Click OK to create the report.

    5. After the report is created the The Calibration Report Options form can be recalled to change report options by selecting Report >> Calibration.
