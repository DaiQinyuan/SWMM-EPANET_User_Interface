Copying a View

EPANET can copy the text and graphics of the current window being viewed to the Windows clipboard or to a file. Views that can be copied in this fashion include the Network Map, graphs, tables, and reports.

To copy the current view to the clipboard or to file:

  1. Select Edit >> Copy To or click on the Standard Toolbar.
  2. Select choices from the Copy dialog box that appears and click the OK button.
  3. If you selected to copy-to-file, enter the name of the file in the Save As dialog box that appears and click OK.
